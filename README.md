## 💡 Overview

<a href="#archiver4">Archiver 4</a> serves Mac users who regularly encounter archive files in formats the built-in tools can't handle, who need to create encrypted archives for secure file delivery, or who want to extract specific files from large archives without the time and storage cost of extracting everything.

The <a href="#archiver4">developer and technical use case</a> involves daily archive handling — source code packages in TAR.GZ and ZIP from repositories, compiled releases in various formats, received project archives that need specific file extraction. Archiver 4's format coverage and preview-before-extract workflow make these operations efficient. The <a href="#archiver4">secure file sharing use case</a> creates password-protected archives for sending sensitive documents — contracts, financial files, confidential project materials — through standard delivery channels where the recipient extracts with any ZIP tool using the provided password.

The <a href="#archiver4">large file management use case</a> uses archive splitting to make large collections sendable through size-limited channels — an asset library that exceeds email attachment limits splits into parts, sent as multiple messages, and reassembled at the destination. <a href="#archiver4">Design and creative professional use case</a> handles the font packs, stock asset collections, and resource bundles that creative work constantly involves — browsing inside archives before extracting, pulling specific assets without unpacking the complete collection.

<a href="#archiver4">Format conversion between archive types</a> opens an archive in one format and re-saves it in a different format — converting a received RAR archive to ZIP for forwarding to someone without RAR support, or converting a 7-Zip archive to TAR for a specific workflow requirement. <a href="#archiver4">Archive integrity verification</a> checks downloaded archives for corruption before extraction — confirming an archive is complete and undamaged without attempting a full extraction that fails partway through.

<a href="#archiver4">One-time purchase from the Mac App Store</a> provides permanent Archiver 4 access — a stable archive utility at a fixed cost without subscription. <a href="#archiver4">Active development</a> maintains Archiver 4 with format compatibility updates and macOS release compatibility as both the archive format landscape and macOS evolve.

## 🤔 FAQ

<details>
<summary>Can Archiver 4 verify archive integrity?</summary>

Yes. Archive integrity checking confirms a downloaded archive is complete and undamaged before extraction.

</details>

<details>
<summary>Does Archiver 4 handle multi-part RAR archives?</summary>

Yes. Multi-part RAR archives extract when all parts are present.

</details>

<details>
<summary>Can I batch extract multiple archives at once?</summary>

Yes. Multiple archives queue for simultaneous batch extraction.

</details>

<details>
<summary>Does Quick Look preview work with Archiver 4?</summary>

Yes. Pressing spacebar on an archive file in Finder shows its contents through Quick Look.

</details>

<details>
<summary>What is the difference between a password-protected archive and Secret Safe?</summary>

A password-protected ZIP is a standard format any tool can open with the password. Secret Safe is Archiver 4's own encrypted container for local secure storage.

</details>
